# 🏍️ Moto-Race

Cyberpunk motorcycle racing game for Claude Code. Race against 4 AI opponents on a neon-lit circuit track.

Built with Three.js, MediaPipe Hands, and `@cloudgeek/glimpse` (native macOS WKWebView host).

## Quick Start

```bash
cd scripts
npm install
node moto-race.mjs
```

## Controls

| Key | Action |
|-----|--------|
| ↑/W | Throttle |
| ↓/S | Brake |
| ←/A | Steer left |
| →/D | Steer right |
| Space | Boost |
| R | Restart |
| Escape | Quit |

Hand gesture controls available when camera is enabled.
